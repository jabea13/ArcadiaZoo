# Arcadia
Ce site est un site vitrine pour le parc zoologique Arcadia.

# Installation

bash
git clone https://github.com/username/ArcadiaZoo
cd ArcadiaZoo
npm install

# Pré-requis

install bootstrap and icon-bootstrap
install symfony and composer

# Usage

npm start
